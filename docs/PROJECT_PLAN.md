# 📋 PROJECT_PLAN · 项目计划

> **目标**：把"小柒去水印插件"从一个"凑合能用的拼装体"，升级为"产品级、可扩展、可复刻"的项目。
> **范围**：基于《新人操作指南》第 4 节的"不足 / 优化 / 扩展 / 复刻"四个象限，按三阶段推进。

---

# 📌 风险与对策

| 风险 | 影响 | 对策 |
|---|---|---|
| 字节系接口持续升级 | 插件静默失效 | 加 `/health-check` 心跳 + 告警 |
