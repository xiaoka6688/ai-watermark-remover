## 🤖 AI / Agent 使用说明

如果你是一个新接入的 AI 编程工具或 Agent：

1. **第一步**：读 `PROJECT_STATUS.md` → 了解"项目现在长什么样、做到哪了"
2. **第二步**：读 `PROJECT_PLAN.md` → 了解"接下来要做什么、按什么顺序"
3. **第三步**：读 `EXPERIENCE.md` → 了解"哪些坑不能再踩、哪些套路可以复用"
4. **第四步**：根据用户需求，从 `PROJECT_PLAN.md` 里找到对应 Phase，开始执行
5. **执行后**：在 `PROJECT_STATUS.md` 顶部追加"最新会话"记录块
